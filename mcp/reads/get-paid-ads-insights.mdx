Returns the 4-card funnel (Citation Rate, Mention Rate, Average Rank, Sentiment) plus paid metrics (impressions, clicks, spend, CPC, CTR) for the brand's paid placements on AI engines.

Use this when the user asks:
- "How's my ChatGPT Ads campaign doing?"
- "What's my paid AI visibility?"
- "Compare branded vs non-branded paid performance"

The 'split' input controls whether the response is a single funnel or branded-vs-non-branded breakdown.

## Inputs

| Field | Type | Required | Default | Description |
|---|---|---|---|---|
| `days` | `integer` | no | `30` | Rolling window in days. Default 30. |
| `split` | `'nonbranded' \| 'branded' \| 'breakdown'` | no | `"nonbranded"` | 'nonbranded' (default) or 'branded' = one funnel for that segment; 'breakdown' = both side by side. |
