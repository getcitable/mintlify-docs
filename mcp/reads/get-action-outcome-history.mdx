Returns recent Citable-emitted actions for the brand, with their measured outcome deltas where the measurement window has resolved. This is the canonical history of "what did Citable do, and what happened next."

Use this when the user asks:
- "What did Citable do for me last month?"
- "Which actions actually moved citations?"
- "Show me the best-performing content actions."
- "What's still measuring (pending outcomes)?"

Skip when the user wants live recommendations (use get_geo_priorities) or current visibility metrics (use get_topical_authority / get_paid_ads_insights).

Inputs:
- actionType (optional): filter by 'content' / 'outreach' / 'technical' / 'operational' / 'paid'
- outcomeStatus (optional): filter by 'pending' (still measuring) / 'measured' (delta computed) / 'inconclusive' / 'failed'
- limit (default 20, max 100)

Response: \{ rows: [...], totalReturned, status, meta \}. Each row has id, actionType, actionSubtype, emittedAt, executedAt, postedUrl, outcomeStatus, deltaMetrics (when measured).

## Inputs

| Field | Type | Required | Default | Description |
|---|---|---|---|---|
| `actionType` | `'content' \| 'outreach' \| 'technical' \| 'operational' \| 'paid'` | no | — | Filter by action class. Omit for all. |
| `outcomeStatus` | `'pending' \| 'measured' \| 'inconclusive' \| 'failed'` | no | — | Filter by outcome state: 'pending' = still measuring, 'measured' = impact computed. |
| `limit` | `integer` | no | `20` | Max rows to return. Default 20, max 100. |
