Returns the brand's topical authority matrix — Citation Rate, Mention Rate, Average Rank, and Sentiment broken down per competitor — filterable by journey stage, persona, and AI model.

Use this when the user asks:
- "Where am I weakest on purchase-stage prompts?"
- "Who's beating me on comparison queries for the founder persona?"
- "How does my Sentiment look across stages?"
- "What's my Citation Rate against [Competitor X]?"

Skip when the user wants a paid-ads-specific funnel (use get_paid_ads_insights) or a to-do list (use get_geo_priorities).

Defaults: 7-day window, non-branded prompts only (isBranded=false), all four stages (awareness_discovery / consideration_evaluation / purchase_intent / support_usage), all personas, all models. Set 'isBranded' to true to query branded prompts — gated to Grow tier; non-Grow callers see own-brand row only with tierGated=true.

Response: \{ citationRate[], mentionRate[], averageRank[], sentiment[], tierGated, scope: \{ stages, isBranded, competitorNames, modelCodes, personaNames, days, appliedTier \} \}. Each metric array has one row per brand (yours + competitors) with isYou flag, brand name, and a numeric value (sentiment has \{ score, sampleSize, confidence \}). Current limitation: per-competitor averageRank/sentiment rows are own-brand-only.

## Inputs

| Field | Type | Required | Default | Description |
|---|---|---|---|---|
| `days` | `integer` | no | — | Rolling window in days. Defaults to 7. |
| `stages` | `'awareness_discovery' \| 'consideration_evaluation' \| 'purchase_intent' \| 'support_usage'[]` | no | — | Filter to specific journey stages. Defaults to all four. |
| `personaNames` | `string[]` | no | — | Filter to specific personas by display name. |
| `modelCodes` | `string[]` | no | — | Filter to specific AI models (e.g. ['chatgpt', 'perplexity']). Defaults to all configured for the brand. |
| `competitorNames` | `string[]` | no | — | Limit comparison to specific competitor names. Defaults to all tracked competitors. |
| `isBranded` | `boolean,null` | no | — | Filter prompts by branded flag. null = both, false = non-branded only (default), true = branded only (gated to Grow tier). |
