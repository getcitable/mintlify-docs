Create or update a paid ad on an AI surface (ChatGPT Ads today; Perplexity / Gemini / Grok / Copilot when they ship).

Use this when the user asks:
- "Launch a ChatGPT Ads campaign for prompt cluster X"
- "Pause ad group Y"
- "Archive that underperforming ad"

The 'engine' input defaults to 'chatgpt'. Every write is validated against the persona's guardrails before it reaches the ad platform (blocking violations halt; warnings are logged). After write, the action is recorded for later outcome measurement.

Skip when the user wants reporting only (use get_paid_ads_insights) or strategy proposals (use recommend_paid_ad_strategy).
